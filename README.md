# Must WatchHub

## Running the Application

1. Open a terminal and navigate to the project folder.

2. Start the Flask application by running:

```
python app.py
```

3. Open a web browser and go to:

```
http://127.0.0.1:5000/
```

The application should now be running.

If you can´t access change the port in app.py. 

## Using the Application

* Browse the list of movies on the front page.
* Search for movies using regular expressions.
* Add movies to your watchlist.
* View your watchlist and update ratings and reviews.

## Example Regex Searches

Movies beginning with "The":

```
^The
```

Movies with genres Action or Drama:

```
Action|Drama
```

Movies containing "Star":

```
Star
```
